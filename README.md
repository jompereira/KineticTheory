# Kinetic Theory — Studio Website

Static marketing site for [Kinetic Theory](https://jompereira.github.io/KineticTheory/), an independent mobile game studio based in Porto, Portugal.

## Pages

| File | URL | Description |
|---|---|---|
| `index.html` | `/` | Home — hero, games grid, about section |
| `privacy-policy.html` | `/privacy-policy.html` | Privacy policy for all games |
| `support.html` | `/support.html` | FAQ and contact |

## Stylesheets

| File | Loaded by |

| `home.css` | `index.html` — hero, games, about, footer |
| `privacy.css` | `privacy-policy.html` |
| `support.css` | `support.html` |
